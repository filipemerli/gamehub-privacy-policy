# GameHub Privacy Policy - GitHub Pages Setup

This repository contains the privacy policy for the GameHub for Roblox Android app.

## Setup Instructions

### 1. Create a GitHub Repository
1. Go to [GitHub](https://github.com) and sign in
2. Click the "+" icon in the top right and select "New repository"
3. Name it something like `gamehub-privacy-policy`
4. Make it **Public**
5. Click "Create repository"

### 2. Upload the Privacy Policy
1. In your new repository, click "Add file" → "Upload files"
2. Upload `privacy-policy.html`
3. **Important:** Rename it to `index.html` (GitHub Pages requires this name)
4. Click "Commit changes"

### 3. Enable GitHub Pages
1. Go to your repository Settings
2. Scroll down to the "Pages" section (in the left sidebar under "Code and automation")
3. Under "Source", select "Deploy from a branch"
4. Under "Branch", select `main` (or `master`) and folder `/ (root)`
5. Click "Save"

### 4. Access Your Privacy Policy
After a few minutes, your privacy policy will be available at:
```
https://[your-username].github.io/gamehub-privacy-policy/
```

## Key Privacy Features

GameHub is built with a privacy-first approach:

- ✅ **No Personal Data Collection** - Zero personal information collected or transmitted
- ✅ **No Tracking or Analytics** - No analytics services or behavior tracking
- ✅ **No Advertisements** - Completely ad-free experience
- ✅ **Local Storage Only** - Watchlist stored locally using SQLite on device
- ✅ **Public APIs Only** - Uses only Roblox's public APIs for game data
- ✅ **No Servers** - No backend servers collecting or storing user data
